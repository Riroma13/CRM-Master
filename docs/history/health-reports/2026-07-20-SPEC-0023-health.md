## Edge Cases Covered

| Case | Coverage |
|------|----------|
| Mid-period upgrade (immediate prorated) | SubscriptionEngine.changePlan() for upgrades |
| Downgrade (next period) | SubscriptionEngine.changePlan() with pendingPlanId |
| Trial → paid conversion (BullMQ saga) | ConvertTrialSaga with compensating actions |
| Trial expiration → expired | LifecycleService.expireTrials() |
| Payment failure → past_due → grace → suspend → cancel | LifecycleService chain |
| Stripe circuit breaker fail-open | StripeCircuitBreaker (3 failures/60s) |
| Webhook replay/idempotency | StripeWebhookProcessor with eventId PK, ON CONFLICT DO NOTHING |
| Webhook timestamp freshness | StripeWebhookGuard rejects events >5 min old |
| Plan limit exceeded (hard) | PlanLimitGuard returns 429 |
| Plan limit warning (soft) | PlanLimitGuard sets X-Limit-Warning header at 80% |

| Risk | Severity | Mitigation |
|------|----------|------------|
